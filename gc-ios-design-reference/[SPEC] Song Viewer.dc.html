Reimagining the song view for leading live

Four arrangements of the same leadsheet — each solving the on-stage problem differently. Chords sit above the syllable they land on (not spaced-out monospace), so the face carries the lyric. Everything is live: tap the key ▾ ▴ to transpose, toggle lyrics-only / section labels / font size, watch the stopwatch tick. Chord display (letters · numbers · solfège) follows the app's Settings — flip it in Tweaks.
