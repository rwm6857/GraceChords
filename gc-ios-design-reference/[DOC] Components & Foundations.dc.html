GraceChords · iOS

What's still missing — foundations & components

The palette, type, icons, app shell, and content patterns are decided. To call this a complete system we still need the connective tissue every native iOS app leans on — the parts that don't show up in a single screen but govern all of them. Below: the gap analysis, then mocks of each missing piece in both modes. Nothing here changes your existing pages.

Part 1 · Foundations
SP

Spacing — 4 pt grid

Today every gap is hand-tuned. A 4-pt ramp keeps screens rhythmic and makes the eventual React Native styles trivial. 16 is the screen margin, 12 the default row gap, 20–24 between groups.
